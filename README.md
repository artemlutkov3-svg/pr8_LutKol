# Лабораторная работа №8

## Генератор паролей

Программа на Python для генерации случайных паролей.

### Возможности:
- Генерация пароля заданной длины
- Поддержка строчных букв
- Поддержка заглавных букв (версия 1.1)
- Поддержка цифр и спецсимволов (версия 1.2)

### Установка и запуск:
1. Убедитесь, что установлен Python
2. Запустите программу командой: "python password.py"
3. Введите желаемую длину пароля
### Пример работы:
Введите длину пароля: 10
Сгенерированный пароль: aB3$dE9#kL

### Версии:
- **ver.1.0** — только строчные буквы
- **ver.1.1** — добавлены заглавные буквы
- **ver.1.2** — добавлены цифры и спецсимволы
